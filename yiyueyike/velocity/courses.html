The Command Line



  • Lesson Notes
  • Troubleshooting
  • Support

The command line can be a scary place for beginners, but there are only a handful of commands you'll need to know in order to use it including pwd, cd, and ls.

1. Open the Command Line

  • On Mac you can go to spotlight "Terminal" and add to your dock.
  • On PC you can search for Gitbash (a program that comes with Rails installer).

2. Customize your Terminal ( Only)

  1. Terminal -> Preferences -> Settings
  2. Select the "Homebrew" theme on the left side
  3. Click "Change" and increase the font size to 18px
  4. Choose "Default"

3. Learn these basic commands


              $ pwd
              $ open . #this is for Mac only. On a Window's PC try "start ." instead of open
              $ ls # Stands for list. Also, this is a comment.
              $ cd Desktop 
              $ cd ~
              $ ls -a

4. Permission denied error? ( Only) 

If you get a permissions error somewhere along the line, you might be able to use "sudo", which is your super user account (aka. it's like the admin of your computer that has all the privileges.) For example:

$ sudo cd Desktop

5. Read Additional Resources (optional)

Zed Shaw's The Command Line Crash Course - this is a great online course and only takes a few hours to get through. If you're short on time I'd recommend you go as far as the lesson on "Moving A File (mv)".

Q: Why isn't "open" working on Windows?

Use "start ." instead. This will work on Windows when you are using the Git Bash command line.

Q: Why isn't "open" working on Ubuntu?

Open isn't a command in Debian systems.

Use "xdg-open" instead. However, "xdg-open ." does not open current directory.
